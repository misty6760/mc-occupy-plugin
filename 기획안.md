# 땅따먹기 기획안

## 1. 맵 환경
      1. 크기: 3600 * 3600
      2. 점령지 5곳: 물, 불, 얼음, 바람, 중앙
      3. 점령지 좌표 모두 공개(중심 좌표로 해야 할 듯)
## 2. 팀 시스템
      1. 팀 수(권장): 3~4팀
      2. 3인 1팀
      3. 스폰 방식: 리스폰 시 하늘에서 스폰
         1. y100 좌표에서 리스폰(느린 낙하 1)
      4. 팀원들끼리 /tpa 사용 가능
## 3. 점령 규칙
      1. 기본 점령지: 물, 불, 얼음, 바람
         1. 한 팀만 점령 중일 때 점령 필요 시간: 5분
         2. 재점령 시 필요 시간: 10분
         3. 각 꼭짓점 점령 시 점수 1점 증가
         4. 점령 시 해당 구역 내에서 효과 발동(설정 되어 있을 때만)
      2. 중앙(특수)
         1. 한 팀만 점령 중일 때 점령 필요 시간: 10분
         2. 재점령 시 필요 시간: 15분
         3. 점령 시 2점 획득
      3. 특수 규칙
         1. 3개의 기본 점령지 점령 시 중앙 점령지 점령 불가
         2. 모든 점령지는 점령된 후 모든 플레이어에게 알림
         3. 특수 점령지는 점령 될 때 채팅으로 전체 알림, 보스바로 점령 현황 표시
## 4. 점령지 주제, 효과
      1. 불
         1. 지형: 용암 지대
         2. 효과: 화염 저항 1레벨 부여(점령한 팀에게만)
      2. 물
         1. 지형: 5칸 높이 물
         2. 수중 호흡 1레벨, 돌고래의 가호 1레벨 부여(점령한 팀에게만)
      3. 얼음
         1. 모든 팀에게 구속 버프 부여(점령 전, 지역 내에서만)
         2. 아군은 면역(점령 후)
      4. 바람
         1. 점프 맵 구조
         2. 신속 1레벨 부여(점령한 팀에게만)
      5. 중앙
         1. 전장
         2. 점령 시 2점 + 2곳
## 5. 승리
      1. 중앙 점령 루트
         1. 중앙(2점) + 기본 점령지(2곳) = 4점 확보
         2. 이후 추가로 점령지 1곳 확보. 총 5점으로 승리
      2. 기본 점령지 루트
         1. 기본 점령지 4곳 확보 시 세트 보너스로 1점 획득
         2. 합계 5점으로 승리
## 6. 교환
      1. 기본 교환 재화: 철 주괴, 청금석
      2. 교환 방식: 왼손 들기
         1. 청금석 64개 -> 경험치 병 64개
         2. 철 32개 -> 빵 64개
         3. 목적: 경험치 및 식량 조달 부담 완화, 전투 및 점령에 집중 유도
